# 🎮 Tic Tac Toe Game

A responsive **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript** with player turns, winner detection, draw detection, and game reset functionality.

---

## 📌 Description

This project is a browser-based implementation of the classic Tic Tac Toe game. Two players can take turns placing **X** and **O** on a 3×3 grid. The game automatically detects the winner, declares a draw when applicable, and allows players to restart the game with a single click.

---

## ✨ Features

- 🎲 Two-player gameplay (X vs O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset game functionality
- 📱 Responsive design
- 🎨 Simple and clean user interface

## 🎮 How to Play

1. Player **O** starts the game.
2. Players take turns placing **O** and **X** on the board.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all nine cells are filled without a winner, the game ends in a draw.
5. Click **Reset Game** to start a new match.
